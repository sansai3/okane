# メール処理アプリ(試作版)

このプロジェクトは、Python バックエンドと JavaScript フロントエンドで動くデスクトップアプリ(後で EXE 化予定)の試作です。まずはメニュー画面とメール処理の最初の機能を実装しています。

## 機能概要

- `config/app_config.json` に記載した6つのボタンがメニューとして表示されます。
- 設定ファイルを書き換えると、ボタンのタイトルや説明、呼び出す Python モジュールを自由に変更できます。
- 1つ目の「メール添付ダウンロード」ボタンでは、メールファイル(.eml/.msg)から添付とパスワード情報を整理します。

## セットアップ

1. 依存パッケージをインストールします。
   ```bash
   pip install -r requirements.txt
   ```
2. 開発用サーバーを起動します。
   ```bash
   python app.py
   ```
3. ブラウザで `http://127.0.0.1:5000` を開きます。

## メール処理の流れ

1. メニューで「メール添付ダウンロード」を選ぶとフォームが開きます。
2. メールファイルが入っているフォルダ、添付を保存するフォルダ、CSVの保存先を入力します。
   - 何も入力しない場合は `config/app_config.json` の `paths` にある既定値が使われます。
3. 実行すると、以下を行います。
   - `.eml`/`.msg` ファイルを読み込み、添付ファイルを `001_ファイル名` のように連番で保存。
   - メール本文からパスワードらしき文字列を抽出。
   - 受信日/送信者/受信者/本文/パスワードを `data/mail_summary.csv` に書き出し。

## 設定のカスタマイズ

- `paths.default_mail_folder` などを変更すると、フォームの初期値が変わります。
- `email_processing.password_keywords` に探したいキーワード(日本語/英語)を追加できます。
- `email_processing.password_symbols_half/full` を変更すると、パスワードに含めたい記号の種類を調整できます。
- ボタンのモジュールを変える場合は `app.buttons[].module` と `entry_point` を編集します。
